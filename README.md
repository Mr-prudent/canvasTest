### 封装锚点类

#### 锚点对象内属性
position // 锚点位置
domObj // 锚点的dom对象
topFunc // 计算top的函数
leftFunc // 计算left的函数

#### 对象所含方法
show // 显示锚点
hide // 隐藏锚点

#### 类的私有属性
pointArr // 保存锚点对象的数组


#### 类的静态方法
showAll // 显示所有的锚点
hideAll // 隐藏所有锚点
togglePoint // 锚点显示切换
pointMouseDown // 锚点点击事件
pointMouseUp
